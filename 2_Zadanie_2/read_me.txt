Lotto Number Game

This is a Python program that simulates a simple lottery game. The program generates six random numbers between 1 and 49
and prompts the user to enter six numbers of their choice. The program then compares the user's numbers to the generated
numbers and displays the number of matching numbers.

Requirements

    Python 3.x
    import random

How to Run

    Install Python on your computer if you don't already have it installed. You can download Python from the official website: https://www.python.org/downloads/

    Save the above code to a file named lotto.py in your desired directory.

    Open a terminal or command prompt and navigate to the directory where you saved the file.

    Run the command python lotto.py in the terminal or command prompt.

    Follow the prompts to enter six numbers between 1 and 49.

    The program will compare your numbers to the generated numbers and display the number of matching numbers.
