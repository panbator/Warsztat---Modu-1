Imagine a Number Game with Flask

This is a simple web application built with Flask that allows the user to play a game where they imagine a number and the computer tries to guess it.
Prerequisites

In order to run this application, you need to have Python and Flask installed on your machine.

How to run

    Clone the repository to your local machine or download and extract the zip file.
    Open a terminal window and navigate to the project directory.
    Run the command python app.py to start the application.
    Open your web browser and go to http://localhost:5000 to play the game.

How to play

    When the game starts, the computer will ask you to imagine a number between 0 and 1000.
    Click on the "Enter" button to continue.
    The computer will guess a number and ask you if it is "Too big", "Too small", or "You win".
    Select one of the three options and click on the "Submit" button.
    The computer will make another guess based on your response.
    Repeat steps 4 and 5 until the computer guesses your number or you reach the maximum number of tries.
    If the computer guesses your number, you win! If you reach the maximum number of tries, the computer will accuse you of cheating.
